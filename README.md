# slides_rnpcv_ufg

Esse repositório contém os slides das aulas ministradas no curso de Redes Neurais Profundas para Visão Computacional (RNP-VC) na Universidade Federal de Goiás (UFG).

Os slides foram feitos por mim (Lucas Araújo Pereira) e foram amplamente inspirados nos slides oferecidos pelo curso cs231n de Stanford (Convolutional Neural Networks for Visual Recognition - http://cs231n.stanford.edu/).

Também foram usados como referência:
Deep Learning Book - http://www.deeplearningbook.org/
Udacity Intro to CV - https://www.udacity.com/course/introduction-to-computer-vision--ud810
Coursera deeplearning.ai DL Specialization - https://www.coursera.org/specializations/deep-learning
Stanford CS236 Deep Generative Models - https://deepgenerativemodels.github.io/
Berkeley CS294-158 Deep Unsupervised Learning - https://sites.google.com/view/berkeley-cs294-158-sp19/home
